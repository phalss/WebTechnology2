FEATURES




  • #1

    DECENTRALIZED

    Traditional money is controlled by banks and governments – which makes it a “centralized” currency. Bitcoin is not controlled or regulated by any single entity like a bank – which makes it a “decentralized” currency.

  • #2

    FAST

    It normally takes just a few minutes for someone on the other side of the world to receive the money as there is no third party involved. On the other hand, traditonal banking transfers can take upto several days. A Bitcoin address can be setup in seconds. Payments are proccesed in real time.

  • #3

    TRANSPARENCY

    Bitcoin stores details of every transaction that ever happened in the network in the blockchain.

  • #4

    DIVISIBILITY

    Old fashioned money can be spent only in amounts as small as a single cent (so up to 2 decimal places). Bitcoin can be spent in much smaller amounts, called Satoshis (all the way up to 8 decimal places). This means that it can be used even for tiny purchases.

  • #5

    LOW FEES

    No transaction fees.
    Minimal conversion fees.

  • #6

    SECURE

    Bitcoin uses cryptography to securely send payments. That’s why Bitcoin is called a cryptocurrency. The code is so strong that tampering is virtually impossible.
